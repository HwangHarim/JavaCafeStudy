> 게임2에서이길확률 은세 번 중두 번 골을 넣을확률과 세 번 모두골을 넣을확률의 합이다   
> P(승리) =5(2,3)+5(3,3)

> P(l, 2어| 성공， 3에 실때)

>  "+" P(1에 실패， 2, 3에 성공)  
>   = p * p * (1 - p) + p * (1 - p) * p + (1 - p) * p * p

> p3 +3 (1-p) p2

### 6-3

> (같은 색이다) 30개의 검은색(또는 흰색) 사각형과， 32개의 흰색(또는 검은색) 사각형이 남는다.

### 6-4

> p( 시계방향) = (1 / 2)3   
> p( 반시계방향) = (1 / 2)3   
> p( 같은 방향) = (1 / 2)3 + (1 / 2)3 = 1 / 4   
> 같은 방향일 때는 충돌 할 수 없기 때문에   
> p( 충돌) = 1 - p(같은 방향) = 1 - 1 / 4 = 3 / 4
